milktea.plonetwitterportlet Installation
----------------------------------------

To install milktea.plonetwitterportlet using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``milktea.plonetwitterportlet`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        milktea.plonetwitterportlet

* Re-run buildout, e.g. with:

    $ ./bin/buildout
